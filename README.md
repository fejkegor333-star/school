# 🗺️ Карта школы - Навигатор

Интерактивная карта школы с построением маршрутов между кабинетами.

## 📁 Файлы

- **map.html** - основная страница навигатора
- **school-map.json** - данные карты (объекты, коридоры)
- **1.png, 2.new.png, 3.png** - планы этажей
- **route-logic.js** - логика построения маршрутов
- **map-data.js** - данные объектов

## 🚀 Использование

### На домене
1. Загрузите все файлы в одну папку на хостинге
2. Откройте `map.html`
3. Карта загрузится автоматически из `school-map.json`

### Локально
1. Откройте `map.html` в браузере
2. Выберите файл `school-map.json` через кнопку

## 🧭 Как пользоваться

1. Выберите точку А (откуда)
2. Выберите точку Б (куда)
3. Нажмите "🗺️ Построить маршрут"
4. Красная линия покажет путь

## 🏗️ Конструктор

Для редактирования карты используйте `index.html` (конструктор).
